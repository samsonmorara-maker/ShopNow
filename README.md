## Shopnow
ShopNow is a modern e-commerce web that allows users to browse products, search items, and manage a shopping cart with smooth
 and responsive user experience.It is built using HTML CSS and Javascript.
## Features
###      products search
 search functinality to filter products by name
###      shopping cart
 Add products to cart
 Automatic cart counter update
###      dynamic products display
Products are rendered dynamically using javascript and easy integration with API
###    responsive design
 works on mobile tablet and desktop devices
## Built by
 HTML5-structure 
 CSS- Styling and layout
Javascript -funtiomality and dom manipulation
##   How it works 
products  are loaded into an array
javascript dynamically displays products on the page
users can search produtss using the search bar and add to cart items
cart counter uodates in real time
